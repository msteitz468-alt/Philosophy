# Road to the Simulation Point (Virk)

> Virk's eleven-stage map of video game technology, from Pong to fully immersive, AI-populated, mind-interfaced ancestor simulations — an engineering argument that the Simulation Point is reachable, not fantastical.

## Summary

The **Simulation Point** is "the point in a technological civilization's development when it possesses the ability to create hyper-realistic simulations... with computing power to keep track of a large number of seemingly individual points of consciousness, record all of their experiences, and render the environment... indistinguishable from what we call the 'physical world.'" The stages argument establishes that forty years of game development (Pong → photorealistic MMORPGs) is a cumulative, exponential trajectory whose remaining steps are extrapolations of existing technology.

## The Eleven Stages

| Stage | Technology |
|---|---|
| 0 | Text adventures — game loop, world as described state (1970s–80s) |
| 1 | Graphical arcade/console games — real-time pixels, physics engines |
| 2 | Graphical adventure/RPG games — persistent characters, inventories, quests |
| 3 | 3D-rendered MMORPGs — persistent shared worlds, avatars, virtual economies |
| 4 | Virtual reality — stereoscopic full immersion |
| 5 | Photorealistic AR/MR — virtual objects overlaid on the physical world |
| 6 | Real-world rendering — light-field displays, holograms, 3D printing |
| 7 | Mind interfaces — bidirectional neural I/O ("mind broadcast") |
| 8 | Implanted memories — false histories for simulated characters |
| 9 | AI and NPCs — characters passing the Turing Test with embodied competence |
| 10 | Downloadable consciousness — digital immortality; consciousness as transferable information |
| 11 | The Simulation Point — ancestor simulations indistinguishable from base reality |

## Key Claims

- **Information-based ontology**: game state (player + world state) is pure information stored outside the rendered world — the model Virk later generalizes to physics and religion.
- **Rendering–physics equivalence**: 3D models, DNA, and light fields all convert stored information into perceptible form; "virtual" vs "physical" becomes a technical, not fundamental, distinction.
- **Procedural generation**: No Man's Sky's 18 quintillion planets show vastness doesn't require manual creation.
- **Persistence without observers**: MMORPG worlds persist as stored information when no player is logged in — a nuance against naive idealism (cf. [[Thinkers/George Berkeley]]'s God as permanent perceiver).
- **Civilizational trajectory**: if we reach the Simulation Point within centuries, older civilizations already have — the setup for [[Arguments/Simulation Argument (Bostrom)]].
- Stage 10 relies on the consciousness-as-information convergence (cf. [[Concepts/Mind Uploading (Kurzweil)]], *Altered Carbon*'s "cortical stacks").

## Contradictions / Open Questions

- > [!warning] Stage 9's assumption that Turing-Test-passing NPCs are achievable engineering is disputed by [[Thinkers/John Searle]] (syntax ≠ semantics) and capability-skeptics like [[Thinkers/Melanie Mitchell]]; Stage 10's uploading assumption inherits every objection filed against [[Concepts/Mind Uploading (Kurzweil)]].
